# Exchange Server End of Support

A comprehensive Jekyll documentation site built with the [Just the Docs](https://just-the-docs.com) theme, covering everything organizations need to plan and execute their Exchange Server migration.

## Topics Covered

- **Checklist** — Phased migration readiness tasks from assessment to decommission
- **Implementation** — Step-by-step guide for Microsoft 365, Exchange Server SE, and Hybrid migration
- **Concerns** — Security, compliance, data sovereignty, business continuity, and user-experience issues
- **TCO Analysis** — 5-year total cost of ownership comparison across migration options
- **Risk Assessment** — Risk register with mitigation strategies for all key risk categories
- **Licensing** — Exchange SE, Microsoft 365 plans, CALs, and licensing decision guide

## View the Site

The site is published at: `https://DekBaCom.github.io/Exchange-Server-End-of-Support/`

## Local Development

```bash
gem install bundler
bundle install
bundle exec jekyll serve
```

Open `http://localhost:4000` in your browser.

- [Jekyll](https://jekyllrb.com/) — Static site generator
